## First steps

1. Review the project overview and tooling areas.
2. Clone the project repositories.
3. Install the recommended development tooling.
4. Run the sample SDV workflow.
5. Join the community and contribute improvements.

## Recommended path

Start with the example workflow, then adapt the templates and validation steps to your local SDV engineering environment.
